HTML系列學習二,HTML編輯器

HTML 编辑器是一种用于创建和编辑 HTML 文档的软件。它们通常提供了许多方便的功能，如语法高亮、代码提示、标签自动闭合等，使得创建和编辑 HTML 文档变得更加方便。

有许多不同的 HTML 编辑器可供选择，包括免费和付费的。例如，Notepad++ 是一款免费的 HTML 编辑器，Adobe Dreamweaver 是一款付费的 HTML 编辑器。这些编辑器都有不同的功能和界面，你可以根据自己的需要选择最适合自己的编辑器。

此外，很多文本编辑器（如 Sublime Text、Atom 等）也可以用来编辑 HTML 文档，尽管它们不是专门为 HTML 设计的，但也可以提供语法高亮、代码提示等功能。

总的来说，选择 HTML 编辑器取决于你的个人喜好和需要。建议你尝试几款不同的 HTML 编辑器，然后选择最适合自己的那一款。

我習慣用VSCode來編輯HTML，Visual Studio Code（简称 VSCode）是一款由 Microsoft 开发的免费开源文本编辑器。VSCode 可以用来编辑各种编程语言的代码，包括 HTML。

VSCode 本身并不是专门为 HTML 设计的编辑器，但它可以通过安装扩展来支持 HTML。安装 HTML 扩展后，VSCode 就可以提供 HTML 代码的语法高亮、标签自动闭合、代码提示等功能。

打开一个新的文件或者编辑已有的 HTML 文件。VSCode 会自动识别文件的类型，并启用 HTML 相关的功能。
你还可以在 VSCode 中使用其他扩展来增强 HTML 的功能，比如 Emmet，它可以让你使用简洁的语法来快速生成 HTML 代码。

总的来说，VSCode 是一款功能强大的文本编辑器，可以用来编辑 HTML 代码。它的扩展机制使得它可以根据你的需要提供丰富的功能，是一个不错的选择。

要在 VSCode 中使用 HTML，你只需要輸入html，你就可以根據選擇提供html的基本樣式。

HTML 文档 = 网页

Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容，第一個基礎實例：

例子解釋

  • <html> 与 </html> 之间的文本描述网页
  • <body> 与 </body> 之间的文本是可见的页面内容
  • <h1> 与 </h1> 之间的文本被显示为标题
  • <p> 与 </p> 之间的文本被显示为段落

HTML 标签

HTML 标签是用于描述 HTML 文档结构和内容的标记。标签通常是由尖括号包围的关键字，比如 <p> 和 </p>。标签通常是成对出现的，开始标签用于描述元素的开始，结束标签用于描述元素的结束。

下面是一些常见的 HTML 标签：

  • <h1> 到 <h6>：用于定义文档中的标题。
  • <p>：用于定义段落。
  • <br>：用于插入换行符。
  • <img>：用于在文档中插入图像。
  • <a>：用于定义超链接。
  • <div>：用于定义文档中的块级元素。
  • <span>：用于定义行内元素。

HTML 标签也可以包含属性，属性是用于提供有关标签的更多信息的附加信息。例如，<img> 标签可以包含 src 属性，用于指定图像的 URL。

Mac本地搭建WordPress

對於網站CMS來說，個人覺得WordPress對於新手最為友好，它是一個內容管理系統 (CMS)，允許用戶輕鬆創建和管理網站。它是一個開源平台，這意味著源代碼可供任何人使用和修改。 WordPress 很受歡迎，因為它易於使用，並且擁有龐大的用戶和開發人員社區，他們為其持續發展做出了貢獻。使用 WordPress，用戶無需知道如何編碼即可創建和管理網站。它提供一系列功能和工具，包括模板和插件，以幫助用戶定制他們的網站並添加聯繫表格、電子商務功能等功能。 WordPress 可用於創建各種各樣的網站，包括博客、商業網站、電子商務商店等。

本地搭建WordPress有兩種方式，方式一是用PHPStudy，方式二是本地安裝LNMP/LAMP，今天先學習用最簡單方便的集成環境搭建，以前用Windows系統的時候用PHPstudy最好用，但是在Mac上就會出現很多問題，例如WordPress需要PHP7.4的版本，PHPstudy的Mac版本卻沒有PHP7.4，因此我選擇了MAMP。

MAMP 是一個集成環境軟件包，可為 macOS 和 Windows 上的 Web 開發人員提供本地開發環境。它代表“Macintosh、Apache、MySQL 和 PHP”，包括以下組件：
Apache：流行的開源網絡服務器軟件
MySQL：流行的開源數據庫管理系統
PHP：一種用於 Web 開發的流行開源編程語言
MAMP 允許開發人員在他們自己的計算機上本地測試他們的網站和應用程序，而無需互聯網連接或實時 Web 服務器。這可用於測試和調試目的，或用於離線開發網站和應用程序。
要使用 MAMP，您需要將其安裝在您的計算機上並設置 Web 服務器和數據庫。然後，您可以使用 MAMP 創建和管理您的 Web 項目，並通過本地計算機上的 Web 瀏覽器訪問它們。

一．什么是Hexo？

Hexo是一个开源的静态博客生成器，它使用Node.js开发，可以快速创建个人博客或网站。Hexo使用Markdown语言编写博客内容，并且提供了丰富的插件和主题来自定义博客的外观和功能。Hexo的优点在于，它的使用非常简单，可以快速生成静态网站，并且有很多主题和插件可供选择，方便博客的自定义。在https://hexo.io/themes/还可以自己选择喜欢的主题框架，我这里就选择最简单的。
