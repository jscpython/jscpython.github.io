如何本地搭建WordPress

對於網站CMS來說，個人覺得WordPress對於新手最為友好，它是一個內容管理系統 (CMS)，允許用戶輕鬆創建和管理網站。它是一個開源平台，這意味著源代碼可供任何人使用和修改。 WordPress 很受歡迎，因為它易於使用，並且擁有龐大的用戶和開發人員社區，他們為其持續發展做出了貢獻。使用 WordPress，用戶無需知道如何編碼即可創建和管理網站。它提供一系列功能和工具，包括模板和插件，以幫助用戶定制他們的網站並添加聯繫表格、電子商務功能等功能。 WordPress 可用於創建各種各樣的網站，包括博客、商業網站、電子商務商店等。

本地搭建WordPress有兩種方式，方式一是用PHPStudy，方式二是本地安裝LNMP/LAMP，今天先學習最簡單方便的PHPStudy。

PHPStudy 是一個免費、開源的 PHP 集成開發環境 (IDE)。它是一個軟件應用程序，提供了一系列用於開發和調試 PHP 代碼的工具和功能。 PHPStudy 包括一個用於編寫代碼的文本編輯器、一個用於查找和修復錯誤的調試器以及一個用於在本地測試代碼的內置 Web 服務器。它還包括對其他語言的支持，例如 HTML、CSS 和 JavaScript。 PHPStudy 旨在易於使用，具有簡單的界面和直接的工具。它經常被想要在本地環境中學習和練習編碼的 PHP 編程初學者使用。

GitHub Pages + Hexo如何搭建个人博客

我是小白程序媛，之前学的东西都是非常碎片化的，哪里不懂找哪里，课程视频也看不少，就是没有真正的记录下来，到下一次要用的时候又把知识点给忘了，又得从新找，浪费很多时间，现在开始，搭建一个自己的博客来记录自己的学习过程。

先从免费搭建个人博客开始，这里我用的是GitHub Pages + Hexo。

一．什么是Hexo？

Hexo是一个开源的静态博客生成器，它使用Node.js开发，可以快速创建个人博客或网站。Hexo使用Markdown语言编写博客内容，并且提供了丰富的插件和主题来自定义博客的外观和功能。Hexo的优点在于，它的使用非常简单，可以快速生成静态网站，并且有很多主题和插件可供选择，方便博客的自定义。在https://hexo.io/themes/还可以自己选择喜欢的主题框架，我这里就选择最简单的。
