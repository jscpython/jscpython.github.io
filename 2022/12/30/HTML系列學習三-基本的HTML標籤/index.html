HTML系列學習三,基本的HTML標籤

HTML (Hypertext Markup Language) 是用來創建網頁的標準標記語言。它使用標籤來描述文檔的結構和內容。標籤一般都以標籤對的形式出現，有開頭就有結束。

以下是一些常用的 HTML 標籤:

  • <html>: 定義 HTML 文檔。

  • <head>: 定義文檔的頭部，其中可以包含腳本和样式信息。

  • <title>: 定義文檔的標題，顯示在瀏覽器的標籤頁中。

  • <body>: 定義文檔的主體，包含文檔的所有內容。

  • <h1> - <h6>: 定義標題。 H1 是最高級別的標題，H6 是最低級別的標題。

  • <p>: 定義段落。

  • <br>: 換行。

  • <a>: 定義超鏈接。

  • <img>: 定義圖像。

  • <div>: 定義文檔中的節。

  • <span>: 定義文本的一小段，用於對文檔中的行內元素進行組合。

  • <ul>: 定義無序列表。

  • <ol>: 定義有序列表。

  • <li>: 定義列表項。

這些標籤可以用來組合成複雜的 HTML 文檔。
