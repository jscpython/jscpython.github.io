作为一名程序员初学者,从头开始,HTML/CSS

以前學的東西都很碎片化，昨天安裝好了WordPress之後才真正認識到，不管是WordPress，其他的CMS,前端，後端,甚至如何開發一個擁有自己需求的CMS系統等等，都需要了解什麼是HTML，什麼是CSS，所以我決定重新開始學習HTML和CSS，這裡我就從最新的HTML5和CSS3開始學。

了解什麼是HTML5？

HTML5是下一代的HTML標準。，它是HTML（HyperText Markup Language）的最新版本。 HTML是一種用於描述網頁內容和結構的語言，它是網頁瀏覽器（如Google Chrome，Firefox等）所使用的語言。 HTML5為網頁提供了更多的標記（tags）和功能，使得網頁的設計和開發更加簡單，並且更加符合現代的網路應用需求。HTML5還支持移動設備，因此在移動設備上瀏覽網頁時可以提供更好的用戶體驗。

多媒體支持：HTML5支持聲音和視頻內容，並且可以在不使用外部插件的情況下嵌入到網頁中。

移動設備支持：HTML5支持移動設備，可以在移動設備上瀏覽網頁。

本地存儲：HTML5允許在用戶的計算機上存儲數據，使得網頁能夠在離線時使用。

語意化：HTML5提供了許多新的元素，使得網頁的意義更加明確，並且使搜索引擎能夠更好地理解網頁的內容。

動態圖形：HTML5支持動態圖形，可以使用JavaScript或Canvas元素創建動態圖形。

多任務支持：HTML5支持多任務，可以在同一個網頁中執行多個任務。

即時通信：HTML5支持即時通信，可以在網頁中實現即時聊天和多人遊戲等功能。

拖放功能：HTML5的拖放功能允許用戶在網頁中拖動元素。這個功能可以用於許多不同的用途，如在網頁上拖動圖片或文件來上傳，或者在網頁中拖動元素來改變它們的順序。

了解什麼是CSS3？

Cascading Style Sheets（CSS）是一种用於描述網頁元素外觀的樣式表語言。它可以用來定義元素的顏色、字體、大小、佈局等樣式屬性。

CSS 可以讓你將樣式和結構分開，這樣你就可以使用相同的 HTML 結構來呈現不同的樣式。例如，你可以使用一份 HTML 文件來定義網頁的結構，然後使用一份 CSS 文件來定義網頁的樣式，這樣就可以通過改變 CSS 文件來改變網頁的整體外觀。

CSS 在網頁開發中非常重要，因為它可以讓你更加精細地控制網頁的樣式，並且能夠有效地組織和管理樣式。 CSS 已經發展到多個版本，包括 CSS1、CSS2 和 CSS3。

以上功能到後期的實戰運用中再一一學習與講解。
