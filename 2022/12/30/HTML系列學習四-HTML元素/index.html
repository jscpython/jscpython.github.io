HTML系列學習四-HTML元素

HTML 元素是 HTML 文檔的構建塊。它們用於構建和格式化網頁的內容。
每個 HTML 元素都由一對開始和結束標記表示，元素的內容位於它們之間。例如，下面的代碼定義了一個段落元素：

開始標籤是<p>，結束標籤是</p>。元素的內容是“This is a paragraph”。
HTML 元素可以具有屬性，這些屬性提供有關該元素的附加信息。屬性在開始標記中指定，通常用於配置元素的行為或外觀。例如，下面的代碼定義了一個帶有 href 屬性的 link 元素，它指定了鏈接的 URL：

有許多不同的 HTML 元素，每個元素都有其特定的用途。一些常見的元素包括：

  • <div>：其他 HTML 元素的容器
  • <h1> - <h6>：不同層次的標題
  • <p>：一段文字
  • <a>：指向另一個網頁的鏈接
  • <img>：圖像
  • <form>：用於收集用戶輸入的表單

有關 HTML 元素及其屬性的完整列表，您可以參考 HTML 規範或使用參考指南，例如 MDN（Mozilla 開發人員網絡）上的 HTML 元素參考。
